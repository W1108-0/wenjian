Windows 隐蔽截屏软件 —— 使用说明

1. 文件清单
   - WinUpdate.exe：主程序，后台截屏并加密保存。
   - DiskView.exe：截图查看器，需授权码登录，可预览、导出、删除截图。
   - SysConfig.exe：配置工具，可修改截屏间隔、监控进程，以及彻底卸载程序。

2. 安装与首次运行
   2.1 文件放置
   - 将三个 .exe 放在任意目录（推荐统一放在一个文件夹，如 C:\Tools\）。
   - 主程序 WinUpdate.exe 可放在隐蔽位置，查看器与配置工具可放在桌面方便使用。

   2.2 首次启动（重要：需管理员权限）
   1. 右键 WinUpdate.exe → 以管理员身份运行（仅首次需要，用于创建开机自启任务）。
   2. 弹出授权窗口，显示 32 位授权码（如 7759S2JMV4UPWIUQY4G4Y0CBBAFDUOKW）。
   3. 务必复制并保存此授权码（它是查看加密截图的唯一凭证）。
   4. 点击“开始使用”，程序将：
      - 加密保存授权码到 %APPDATA%\ScreenCaptureApp\
      - 创建 Windows 计划任务 WindowsUpdateTask（登录时自动以最高权限隐藏运行 WinUpdate.exe）
   5. 窗口关闭后，程序转入后台运行，任务栏无图标。

   > 注意：若未以管理员身份运行，会弹出警告“创建开机自启任务失败”，但程序仍可手动启动。此时开机自启不会生效，建议重新以管理员身份运行一次。

3. 后台运行与截屏机制
   - 截屏存储：C:\Users\你的用户名\AppData\Roaming\ScreenCaptureApp\captures\，文件格式 YYYYMMDD_HHMMSS.enc，已 AES-256 加密。
   - 默认间隔：60 秒截屏一次（可在 SysConfig 中修改，最小 5 秒）。
   - 进程触发：若配置了监控进程名（如 chrome.exe），则仅当该进程运行时才截屏；留空则始终截屏。
   - 文件隐藏：所有程序及数据目录均设为“系统+隐藏”属性，需在文件夹选项中显示隐藏文件并取消隐藏受保护的系统文件才能看到。

4. 查看与管理截图（DiskView）
   1. 双击 DiskView.exe，输入之前保存的授权码。
   2. 左侧列表按时间倒序显示所有截图，双击即可在右侧预览原图（可滚动查看）。
   3. 多选导出：按住 Ctrl 或 Shift 选择文件 → 点“导出选中” → 选择保存目录 → 解密为 .jpg。
   4. 多选删除：选中文件 → 点“删除选中” → 确认后永久删除。
   5. 刷新列表：点击可更新最新截图。

5. 修改配置（SysConfig）
   1. 双击 SysConfig.exe，输入授权码。
   2. 常规设置页：
      - 截屏间隔（秒）：输入整数（≥5），点“保存设置”。
      - 监控进程名：多个用英文逗号分隔，留空表示始终截屏；可通过“从文件选择”快速填入。
   3. 系统维护页：
      - 卸载：彻底移除软件（见下节）。
      - 添加到Defender排除项：需管理员权限，避免杀软误删。
      - 强力清理：清除注册表、计划任务等所有残留启动项（用于卸载失败时补救）。

6. 彻底卸载
   1. 运行 SysConfig.exe → 切换到“系统维护” → 点击卸载。
   2. 依次选择是否删除截图和配置数据。
   3. 若弹出 UAC 窗口，点击“是”以完成进程终止和计划任务删除。
   4. 等待提示“卸载完成”，手动关闭配置窗口。
   5. 若自动删除失败，按以下步骤手动清理：
      - 打开命令提示符（管理员）执行：
           taskkill /f /im WinUpdate.exe
      - 删除文件夹 %APPDATA%\ScreenCaptureApp
      - 删除注册表项：HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run 中的 WindowsUpdate（若存在）
      - 删除计划任务：schtasks /delete /tn WindowsUpdateTask /f

7. 常见问题排查
   | 现象                                | 解决方法                                                                 |
   |-------------------------------------|--------------------------------------------------------------------------|
   | 双击 WinUpdate.exe 无反应           | ① 被杀软拦截，添加信任或暂时关闭实时保护；② 检查 %APPDATA%\ScreenCaptureApp\error.log；③ 以管理员身份运行命令提示符，执行 WinUpdate.exe 查看错误信息。 |
   | 截图列表为空                        | ① 检查 SysConfig 中监控进程设置是否正确；② 确认截屏条件是否满足；③ 查看日志文件是否有 mss 截屏错误。 |
   | DiskView 预览失败                   | 授权码输入错误或 auth.dat 损坏，请确保授权码正确。                      |
   | 无法添加 Defender 排除项            | 右键 SysConfig.exe 以管理员身份运行。                                    |
   | 开机不自启                          | 检查计划任务 WindowsUpdateTask 是否存在（运行 taskschd.msc 查看）。若不存在，请以管理员身份重新运行 WinUpdate.exe 一次。 |

> 系统要求：Windows 10/11 所有版本（家庭版可用），无需安装额外运行库。防截屏功能需 1809 以上版本。
> 警告：请合法使用本软件，仅供授权的系统监控与安全审计。
